## 📝 Описание заданий

1.  **Первая программа с переменными:** Объявление и инициализация переменных `int`, `float`, `char` с последующим выводом на экран.
2.  **Размеры типов:** Использование оператора `sizeof()` для определения размера базовых типов данных в байтах.
3.  **Переменные и память:** Вывод значений переменных и их адресов в памяти (в шестнадцатеричном формате) для демонстрации расположения данных.
4.  **Остаток от деления:** Программа считывает два целых числа и выводит остаток от деления первого на второе с помощью оператора `%`.
5.  **Инкремент и декремент:** Демонстрация разницы между префиксной и постфиксной формами операторов `++` и `--`.
6.  **Преобразование типов:** Иллюстрация целочисленного деления и способы получения вещественного результата (2.5) путем приведения типов.
7.  **Потеря точности:** Вывод результата сложения чисел с плавающей точкой (0.1 + 0.2) с высокой точностью (20 знаков) для демонстрации погрешности двоичного представления.
8.  **Переполнение:** Прибавление 1 к максимальному значению `int` (2147483647) и анализ полученного отрицательного числа (переполнение знакового целого).
9.  **Вычисление формул:** Считывание числа `x` и вычисление значений выражений `x² + 2x + 1` и `(x + 1)² - (x - 1)²`.
10. **Обмен переменных:** Классический алгоритм обмена значениями двух переменных с использованием третьей (временной) переменной.
11. **Последняя цифра числа:** Получение последней цифры целого числа с использованием оператора взятия остатка `% 10`.
12. **Разложение числа:** Разделение трёхзначного числа на цифры (сотни, десятки, единицы) с помощью операций деления и взятия остатка.
13. **Формат времени:** Перевод количества секунд, прошедших с начала суток, в формат часы:минуты:секунды (`HH:MM:SS`).
14. **Исследование памяти:** Вывод адресов трех последовательно объявленных переменных `int` и вычисление разницы между ними для анализа особенностей выравнивания данных в стеке.
